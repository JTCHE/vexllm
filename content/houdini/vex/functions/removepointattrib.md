# removepointattrib VEX function

> Removes a point attribute from a geometry.

If you don’t know the class of attribute until runtime, use [removeattrib](https://undefined/docs/houdini/vex/functions/removeattrib).

`int  removepointattrib(int geohandle, string name)`

Removes a point attribute from the given geometry.
